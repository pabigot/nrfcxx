Detailed Description

Structure that can be broadcast as a beacon to provide observation data.

Note
It is intended that the non-negative value of status be used as the sd::Beacon::frame_prefix_s::flags value so the drive mode associated with baseline can be identified.

int16_t nrfcxx::sensor::ccs811::observation_beacon_type::status

Untransmitted content to simplify change detection.

When negative this indicates that the source observation was not ready, and so the beacon must not be transmitted. A non-negative value conveys the current retained_state_type::flags value in its low byte, which is copied into the beacon frame_prefix_s::flags field for transmission.
