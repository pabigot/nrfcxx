Detailed Description

Captured results after an observation.

If ST_DATA_READY is not set in status then the eCO2, eTVOC, and baseline values are not valid.

Field Documentation

◆ eTVOC

uint16_t nrfcxx::sensor::ccs811::observations_type::eTVOC

Equivalent total volatile organic compound estimate, in ppb.

Note
This value appears to be derived from eCO2 or vice-versa. The exact relationship is undocumented, but it appears that 1 eTVOC corresponds to about 7 eCO2.
