uint8_t nrfcxx::periph::GPIOTE::sense_status_type::counter_state

Pin state and count of confirmed changes.

Bit 0 reflects the state of the pin at the time the event was processed. It may have changed subseqently; if so, it is guaranteed that another sense event will be generated.

Bits 1..7 provide a count of confirmed changes since the last sense change event.
