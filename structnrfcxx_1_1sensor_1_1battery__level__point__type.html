Detailed Description

A point in a battery discharge curve sequence.

A discharge curve is defined as a sequence of these points, where the first point has lvl_pptt set to 10000 and the last point has lvl_pptt set to zero. Both lvl_pptt and lvl_mV should be monotonic decreasing within the sequence.
