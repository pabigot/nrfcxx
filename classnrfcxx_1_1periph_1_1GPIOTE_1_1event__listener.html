Detailed Description

Object used to manage event callbacks.

This object provides a callback and argument to be invoked from the GPIOTE IRQ handler in situations where an IN event is triggered for a instance. Association of the listener with the instance is maintained by enable() and disable() regardless of the mode of the GPIOTE instance or whether event interrupts are enabled. The association is implicitly removed if the instance is destructed or enable() is invoked for the same GPIOTE instance but a different listener.
