Detailed Description

Beacon providing system state information.

This basically exposes all the important information from systemState::state_type captured at boot, along with the relative heap and stack use.

This beacon is configured with:

  • min_interval_utt() set to 1 s
  • max_interval_utt() set to 60 min

It is not expected that the beacon interval will be reset, nor that the content of the beacon will change after a state state reached within the first few minutes after reset.
