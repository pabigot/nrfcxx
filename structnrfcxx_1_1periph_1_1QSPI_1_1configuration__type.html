Detailed Description

Configuration parameters for QSPI peripheral.

This structure default-initializes to the nRF52840 power-up default values, except for DPMDUR which uses a shorter maximum delay.

◆ ifconfig

uint32_t nrfcxx::periph::QSPI::configuration_type::ifconfig[2] = {0, 0x40480}

The values for the IFCONFIG0 and IFCONFIG1 registers.

Note
If IFCONFIG0.READOC or IFCONFIG0.WRITEOC indicates that IO2 and IO3 are required for read or write operations activate() will attempt to ensure the SR.QE field in the device is set before it returns.
Do not set IFCONFIG1.DPMEN to Enter here. The implementation assumes the configured value is Exit.
